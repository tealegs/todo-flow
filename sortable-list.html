Drag and Drop items with Modifiable Text

by Tom.

(thx to BS4 + Daemonite Material Design + Sortable.js)

Plugin for BS4 to be able to drag and drop elements with the screen. Sortable.js does all the job without jQuery.

Tutorial

  1. Add this lines to your html, below boostrap.bundle.js
    <script src="https://cdn.jsdelivr.net/gh/RubaXa/Sortable/Sortable.min.js">
  2. Initialize Sortable
    Sortable.create(demo1, { animation: 100, group: 'list-1', draggable: '.list-group-item', handle: '.list-group-item', sort: true, filter: '.sortable-disabled', chosenClass: 'active' });
  3. Read full documentation
